Minecraft Server Info

When you signup create a server and and choose the options on the egg that you would like. If you ever have any questions please join the discord server and request help there. We have a dedicated support team there for your every need! We have anything from java to bedrock to modded servers!

Discord Bot Info

On signup on the client panel create a server. It will be the same egg as all the others because it is a multi egg. Now when the server is created go to the control panel and choose the options for what kind of discord bot you have and then you can start hosting! If you ever have any questions feel free to join the discord server and create a ticket for support from our dedicated support team!

GTA Info

When you signup and create a server you can also create a GTA server with our hosting and our egg. We have regular GTA servers and we also have modded GTA servers for you to choose from. When you select the kind of server you want it will install and you can enjoy it. If you ever have any questions or are having trouble creating a server please join the discord server and you can chat with our dedicated support team!

We also have many other servers!

We also have many other kinds of servers that you can see when you install a server and look at all of the options in the egg we have provided! We have so many servers you won't even be able to use them all :) When you ahve created a server and you play on it if you are ever done with that specific game you can go to the client panel and delete the server and change it or change its resources. You can also reinstall the server from the control panel if you choose to do that! If you ever have any problems or questions don't hesitate to reach out to our dedicated support team on discord!
